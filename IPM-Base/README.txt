The files in this repository are the R code and data used to conduct the analysis for:

Moore JL, Lipcius RN, Puckett B, Schreiber, S. The demographic consequences of growing older and bigger in oyster populations. Ecological Applications, 26(7):2206-2217. 2016.
https://doi.org/10.1002/eap.1374

For a full description of the code and data, see oysterIPM_Metadata.pdf.
