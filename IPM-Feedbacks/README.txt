The files in this repository are the R code and data used to conduct the analysis for:

Moore JL, Puckett B, Schreiber SJ. Restoration of Eastern oyster populations with positive density dependence. Ecological Applications, 28(4):897-909. 2018.
https://doi.org/10.1002/eap.1694

For a full description of the code and data, see oysterFeedbacks_Metadata.pdf.
