The files in this repository are the R code and data used to conduct the analysis for:

Schreiber SJ, Moore JL. The structured demography of open populations in fluctuating environments. Methods in Ecology and Evolution, 9:1569-1580. 2018.
https://doi.org/10.1111/2041-210X.12991

For a full description of the code and data, see oysterStochastic_Metadata.pdf.
